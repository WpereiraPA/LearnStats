## ðŸŒŸ Overview

**LearnStats** is an R package designed to make statistical analysis **accessible**, **interactive**, and **educational**.  
It provides bilingual (English and Portuguese) statistical tools with step-by-step guidance, making it ideal for:

- Students learning statistics  
- Educators teaching statistical methods  
- Researchers seeking quick interactive analyses  
- Portuguese-speaking users who prefer their native language
